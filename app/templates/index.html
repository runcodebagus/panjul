Tentang Aplikasi

ðŸŽ¯ Tujuan Aplikasi

Aplikasi ini bertujuan untuk membantu pengguna dalam menganalisis sentimen dari teks, baik dari ulasan, media sosial, maupun data survei. Dengan pendekatan machine learning, pengguna bisa mengetahui opini publik dengan cepat dan akurat.

Cara Kerja

1. Masukkan teks secara langsung atau unggah file berisi data.
2. Masuk ke menu analisis jika ingin langsung analisis sentimen
3. Jika ada data dalam bentuk file excel dan csv, pilih menu upload untuk mengunggah data.
4. Aplikasi akan memproses data menggunakan model Naive Bayes untuk menentukan sentimen.
5. Hasil analisis akan ditampilkan dalam bentuk grafik dan ringkasan sentimen.

Teknologi yang Digunakan

- Python (Flask)
- Naive Bayes
- HTML, CSS, Bootstrap

Tentang Pengembang

Aplikasi ini dikembangkan oleh Sofanji, mahasiswa Politeknik Harapan Bersama Tegal, sebagai bagian dari tugas akhir. Fokus pada bidang NLP, data science, dan pengembangan web.
